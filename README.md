# Ejercicios JavaScript

Prácticas de Programación Orientada a Objetos (POO) con JavaScript.

## Descripción

Este proyecto reúne una serie de ejercicios prácticos para trabajar conceptos básicos de POO, como:

- creación de objetos
- definición de propiedades y métodos
- manipulación del estado de los objetos
- uso de arrays con objetos
- implementación de lógica simple con JavaScript

## Ejercicios incluidos

1. Crear un objeto "auto" con métodos para encender y apagar.
2. Implementar una "cuenta bancaria" con ingreso, extracción e información del saldo.
3. Crear objetos "rectángulo" con cálculo de perímetro y área.
4. Definir objetos "producto" y almacenarlos en un array.
5. Crear un objeto "persona" con datos, generación, mayoría de edad y generación de DNI.
6. Desarrollar un objeto "libro" con método para mostrar su información.
7. Construir una agenda telefónica con operaciones básicas como agregar, buscar, eliminar y listar contactos.

## Estructura del proyecto

## Cómo usar

1. Abrir el archivo index.html en el navegador.
2. Ejecutar las prácticas desde los archivos JavaScript correspondientes.
3. Seguir las consignas de cada ejercicio desde la consola o la interfaz.
